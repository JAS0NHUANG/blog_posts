# Cloud Computing Characteristics
- On-demand self service
- Global, multi-platform access
- Multi-tenancy and resource pooling
- Elasticity and Scalability
- Mesured

## Advantages
- Trade Capital expense(CAPEX) for operational expense(OPEX)
- economies of scale(bulk price)
- No guessing capacity
- Speed and agility
- No PHYSICAL data centers to maintain

## Types
- IaaS (networking, VMs, data storage...)
AWS EC2, GCP, Azure....etc

- PaaS (customer cab focus on deployment and management of APPs)
AWS Elastic Beanstalk, GCP App Engine, Heroku

- Saas (APP as service)
AWS, Gmail, Dropbox...etc

## AWS Pricing fundamentals
Compute: compute time
Storage: data stored
Data transfer: data transfer OUT (IN is free)
